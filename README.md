# Cronus UI

## Release

`$ pnpm version patch`

`$ git push origin main --follow-tags`

`$ pnpm release`

## Development

- Install dependencies:

```bash
pnpm install
```

- Run the playground:

```bash
pnpm run playground
```

- Run the unit tests:

```bash
pnpm run test
```

- Build the library:

```bash
pnpm run build
```
