# Cronus UI

> A simple UI library for my personal projects. [Storybook](https://mesalilac.github.io/cronus-ui)

## Release

`$ pnpm version patch`

`$ ./push-tag.sh`

## Development

- Install dependencies:

```bash
pnpm install
```

- Run the playground:

```bash
pnpm run playground
```

- Run the unit tests:

```bash
pnpm run test
```

- Build the library:

```bash
pnpm run build
```
